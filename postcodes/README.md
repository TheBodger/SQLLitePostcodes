Postcodes should be downloaded and unzipped from the ordance survey open data portal. 
The data is available in a variety of formats, including CSV, JSON, and XML. 
The data is updated regularly, so it is important to check for updates before using the data.

https://osdatahub.os.uk/data/downloads/open/CodePointOpen

select csv format

unzip the file and copy the contents of the data/csv folder into the Postcodes folder in this project. You should see multiple files, starting with the first 2 letters of the postcode they contain (i.e. ab.csv, b.csv etc)
